**对应解读**

### 公式 (2)

**代码（最底层实现）**

链接：  
https://github.com/xingzhiyuan1229/Experimental_PANN_learning/blob/main/PANN_lib/NN_3D_models.py#L38-L45

```python
# Compute the right Cauchy-Green tensor C = F^T * F
C = tf.linalg.matmul(tf.linalg.matrix_transpose(x), x)
I1 = tf.linalg.trace(C)
I2 = 0.5*(tf.math.square(tf.linalg.trace(C))-tf.linalg.trace(tf.linalg.matmul(C,C)))
I3 = tf.linalg.det(C)
J  = tf.linalg.det(x)

return tf.stack([I1, I2, I3, -2*J], axis=-1)
```

**公式**

\[
I_1 = \mathrm{tr}(\mathbf{C}), \quad I_2 = \mathrm{tr}(\mathrm{cof}(\mathbf{C})), \quad I_3 = \det(\mathbf{C}) = J^2.
\]

**对应解读**  
代码直接计算 \(I_1, I_3, J\)；\(I_2\) 用 \( \tfrac12\left[(\mathrm{tr}\mathbf{C})^2-\mathrm{tr}(\mathbf{C}^2)\right]\) 的等价形式实现。额外返回 `-2*J` 对应文中补充不变量 \(I_1^*=-2J\)。

**公式**

\[
\eta = 2\left( \frac{\partial W_{\mathrm{ICNN}}}{\partial I_1}
+ 2 \frac{\partial W_{\mathrm{ICNN}}}{\partial I_2}
+ \frac{\partial W_{\mathrm{ICNN}}}{\partial I_3}

```python
def matrix_log(C):
    eigenvalues, eigenvectors = tf.linalg.eigh(C)
    log_eigenvalues = tf.math.log(eigenvalues)
    log_diag = tf.linalg.diag(log_eigenvalues)
    log_C = tf.matmul(eigenvectors, tf.matmul(log_diag, tf.linalg.matrix_transpose(eigenvectors)))
    
    return log_C
```

```python
C = tf.linalg.matmul(x2,tf.linalg.matrix_transpose(x2))
H = 0.5 * matrix_log(C)
```

**公式**

\[

\]

**对应解读**  
`matrix_log` + `H = 0.5 * matrix_log(C)` 直接实现 Hencky 对数应变张量。

devH =  H - (1/3) * trace_H * tf.eye(3, dtype=H.dtype)

\end{array}
\]
